CSS FlexBox

Task 6 - Learn CSS Flexbox by Building a Photo Gallery

This is an exercise to show the learning and understanding of the CSS FlexBox Model by building a responsive photo gallery.

Areas covered:

    HTML

  • - re-typing out the boilerplate for good practise and revisiting the html tag
  • - re-learning and typing out the meta tag viewport and encoding settings

    • CSS

  • - More Global Normalisation settings with box-sizing
  • - Transorming text from lowercase to uppercase with text-transform
  • - Re-learning border-bottom shorthand
  • - Understanding the parent - child container roles in the Flexbox Model
  • - Learning about item distribution and arrangement
  • - Balancing image sizes and understaing distortion fixes
  • - Understanding the Flexbox alignment and responsive stacking order

See the Pen HTML CSS | FCC | Task 6 - Learn CSS Flexbox by Building a Photo Gallery by El Rio (@elriomedia) on CodePen.

Marker Pens

Task 3 - Learn CSS By Building A Set Of Coloured Marker Pens

For this task we built 3 different coloured marker pens to give us an introduction into CSS colours, colour theory and how they can be implemented.

HTML - The pens are built by layering divs and using the inline-block positioning to move elements accordingly.

List of areas covered and learned:

Colour Theory:

  • - Primary, secondary and tertiary Colours
  • - Complementary colours, hue, colour wheel, colour theory, contrast, accent, opacity (wall=opaque, glass=transparent), colour models (additive RGB and subtractive CMYK)

CSS Colours:

  • - Keywords, RGB(A), HEX, HSL(A)
  • - Alpha channel
  • - Positioning - Centring with margin: auto (must have width set)
  • - Display Property - display: inline, block, inline-block
  • - Border Property - border: width style color;
  • - Box-Shadow Property - box-shadow: offsetX offsetY blurRadius spreadRadius color; (blurRadius and spreadRadius optional and default to 0)
  • - Gradients - background: linear-gradient(direction, colour1, colour2, and moreâ€¦)
