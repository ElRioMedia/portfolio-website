Switching Variable Contents Test. (Understanding of Variables Explanation)

Lesson 105

How do you replace the contents of a and b?

For this test we had to switch around the contents of the variables a and b, so a = 8 and b = 3.
Rules for the test:
No re-declaring of existing variables, no typing number values, no adjusting original code. 

          function test() {
              var a = "3";
              var b = "8";
          /***********Do not change the code above 👆*******/

          //Write your code on lines 7 - 9:

          /***********Do not change the code below 👇*******/
              console.log("a is " + a);
              console.log("b is " + b);
          }

For my logic I thought about her analogy of replacing the bucket of sand with a bucket of water. Logically you cannot switch these in any way by pouring one into the other. You need to empty one of the buckets first, and then replace it with the something else.
So here I have created declared a new variable name 'c', which is where I chose to store the contents of 'a', then I can now safely replace 'a' with 'b', and then 'b' with 'c' (originally 'a'). See inline comments below in solution. 

        function test() {
            var a = "3";
            var b = "8";
        /***********Do not change the code above 👆*******/

        //Write your code on lines 7 - 9:
        var c = a; //store 'a' (3) in a new variable name 'c' (store contents in another place)
        a = b; //now replace 'a' (3) with 'b' (8)
        b = c; //now replace 'b' (8) with the value of 3, which we stored in new var 'c'

        /***********Do not change the code below 👇*******/
            console.log("a is " + a);
            console.log("b is " + b);
        }
